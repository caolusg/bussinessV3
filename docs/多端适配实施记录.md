# 多端适配实施记录

Last updated: 2026-05-18

## 目标

让商务中文 AI 模拟系统可以在手机、平板、不同尺寸电脑和主流浏览器中完成核心流程。

第一阶段优先保证：

- 学生可以在手机端登录、完善档案、选择业务阶段、查看学习资源。
- 学生可以在手机和平板端进入模拟谈判，发送消息、上传资料、查看练习目标和 AI 教练反馈。
- 教师后台在小屏设备上至少可以切换模块、查看系统数据和基础管理页面。
- 生产构建不再依赖浏览器运行时加载 Tailwind CDN。

## 本轮已完成

### 1. 构建基线

- 补齐本地 `apps/web` 与 `apps/api` 依赖。
- 已验证：
  - `npm.cmd run web:build`
  - `npm.cmd run api:build`

### 2. 前端样式链路

- `apps/web/index.html` 已移除 Tailwind CDN、Google Fonts 外链和旧 importmap。
- 新增 `apps/web/index.css`，通过 Vite 构建 Tailwind CSS。
- `apps/web/vite.config.ts` 已接入 `@tailwindcss/vite`。
- `apps/web/index.tsx` 已导入 `index.css`。

### 3. 学生端工作台

- 桌面端继续使用左侧学习资源栏。
- 手机/平板端隐藏固定侧栏，新增顶部“学习资源库”横向快捷区。
- 工作流地图在小屏下改为 4 列网格，在中大屏保留横向业务流程地图。
- 主内容区改为响应式宽度，避免 `20%` 固定侧栏挤压手机页面。

### 4. 模拟谈判页

- 页面高度从 `100vh` 调整为 `100dvh`，减少移动浏览器地址栏导致的遮挡问题。
- 手机端新增业务阶段横向切换条。
- 手机/平板端新增“练习目标、AI 教练与支持工具”折叠面板。
- 宽屏端继续展示右侧 AI 教练与练习目标面板。
- 输入区在手机端改为纵向布局，避免操作按钮压缩输入框。

### 5. 教师后台

- 小屏下隐藏固定左侧后台导航，新增顶部横向模块切换条。
- 系统数据页由固定两列改为小屏单列、宽屏两列。
- 系统数据筛选工具栏支持换行，表格继续保留横向滚动。

### 6. 档案表单

- 学生档案与密码修改表单从固定两列改为手机单列、平板以上两列。
